Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language designed for creating network-centric applications.

Inside a host environment, JavaScript can be connected to the objects of its environment to provide programmatic control over them.

console.log("Welcome to JavaScript!");
Prerequisites

Before diving deep into programming core concepts, you should possess a functional understanding of the following structural building blocks:

  • A general understanding of how the World Wide Web operates.
  • Good familiarity with HyperText Markup Language (HTML).
  • Basic proficiency with Cascading Style Sheets (CSS).
  • A modern, updated web browser interface.
  • A clean environment text-editing software.

Having prior background knowledge in structural or procedural programming concepts will accelerate your learning curve significantly.

Hello World

To get started with writing your very first code syntax, you can construct a basic execution line within your development workspace.

The standard convention across software engineering involves invoking a program engine designed to instantly print textual notifications.

alert("Hello World");
Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain strict rules.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9).

let developerName = "Alex";
Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task.

To use a function, you must define it somewhere in the scope from which you wish to call or execute it.

function displayGreeting() { return "Hello!"; }
